# wxai
微信AI聊天，可根据人替换prompt

安装环境requirement.txt

使用控制台来到你的python环境
使用命令pip install -r requirements.txt
等待安装完成即可
在t1程序中输入deepseek的api
在t1程序中listen_list = ['','']输入你要监听的人
在t1程序的sender_list[]中输入你在监听群中要监听的不同人的名字，可以定制不同的prompt回话
在每个message的content中输入针对每个人的prompt
打开微信，启动程序
监听列表中的任何人发送on，ai就会启动
发送off，ai就会关闭
使用即可
